## 📸 PicHunt

A React application that allows users to search and browse images from the [Pexels API](https://www.pexels.com/api/).

### 🌐 [Live Demo](https://jsnata.github.io/picHunt/)

## 🚀 Features

- Image Search
- Infinite scroll
- Light/Dark Theme Toggle
- Favorite Images
- Responsive Grid Layout

## 🛠️ Technologies Used

- React
- React Query
- Material-UI (MUI)
- Pexels API
